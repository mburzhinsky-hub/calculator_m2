# ВИЖУ — калькулятор оценки проектов

Одностраничный веб-калькулятор для предварительной оценки мультимедийных и интерактивных проектов ВИЖУ.

Калькулятор помогает быстро собрать бриф, выбрать параметры площадки, зоны, специальные пространства и дополнительные опции, а затем получить ориентировочную стоимость, структуру предложения и выгрузку результата.

## Как запустить

Вариант 1 — локально:

1. Скачайте файл `index.html`.
2. Откройте его двойным кликом в браузере.
3. Заполните форму и скачайте результат.

Вариант 2 — через GitHub Pages:

1. Создайте репозиторий на GitHub.
2. Загрузите в него `index.html` и `README.md`.
3. Откройте настройки репозитория: **Settings → Pages**.
4. В разделе **Build and deployment** выберите:
   - Source: **Deploy from a branch**;
   - Branch: **main**;
   - Folder: **/root**.
5. Сохраните настройки и дождитесь публикации сайта.

## Как обновлять коэффициенты

Все тарифы, коэффициенты и справочники находятся внутри `index.html` в объекте `CONFIG`.

Чтобы изменить расчёт:

1. Откройте `index.html` в редакторе кода.
2. Найдите строку `const CONFIG = {`.
3. Измените нужные значения:
   - `finishRates` — стоимость отделки за м²;
   - `installationTypes` — стоимость простой, средней и большой инсталляции;
   - `zoneTypes` — состав малой, средней и крупной зоны;
   - `segmentShares` — доли контента, оборудования и застройки;
   - `wowZones` — специальные пространства;
   - `options` — мобильное приложение, система управления и поддержка;
   - `coefficients` — коэффициенты высоты и срочности;
   - `logistics` — логистика;
   - `proposalPercents` — статьи предложения;
   - `vat` — ставка НДС.
4. Сохраните файл и обновите страницу в браузере.

Не меняйте функции расчёта, если нужно только обновить ставки или проценты.

## Что умеет калькулятор

- пошаговый ввод данных;
- автоматический пересчёт стоимости;
- проверка обязательных полей;
- детализация по основным блокам;
- детализация по зонам и инсталляциям;
- копирование результата;
- скачивание текстового отчёта;
- скачивание стилизованной таблицы Excel;
- работа без сервера, сборки и внешних библиотек.


## Установка на iPhone как приложение

1. Откройте опубликованную страницу GitHub Pages в Safari на iPhone.
2. Нажмите кнопку **Поделиться**.
3. Выберите **На экран «Домой»**.
4. Подтвердите название, например `ВИЖУ`.
5. На рабочем столе появится иконка калькулятора, а страница будет открываться в полноэкранном режиме как веб-приложение.

Для корректной иконки в репозитории должны лежать файлы `apple-touch-icon.png`, `icon-192.png`, `icon-512.png` и `manifest.webmanifest` рядом с `index.html`.

## Публикация обновлений

После изменения `index.html` загрузите новую версию файла в репозиторий. GitHub Pages автоматически обновит опубликованную страницу через несколько минут.
